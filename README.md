# Сообщения в консоли AG Grid

При запуске в npm run dev могут отображаться сообщения 'AG Grid Enterprise License', 'License Key Not Found' и 'All AG Grid Enterprise features are unlocked for trial'

Это не ошибка бизнес-логики и не поломка приложения. Чтобы убрать эти сообщения и watermark, нужен валидный ключ AG Grid.
